## summary
Educational Ethereum lottery DApp built for a university blockchain course. Demonstrates Solidity smart-contract deployment, Sepolia testnet interaction, and a simple React/Web3 frontend. Not intended for production use; randomness is simplified for learning purposes.
### DEV
deploy contract with remix in sepolia (/src/contract/). Change the contract address in lottery.js (/src/contract/lottery.js), install dependencies and do npm run start
